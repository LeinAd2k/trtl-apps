Getting started

Once you have created an app in the console you are ready to get started using TurtleCoin in your app. Listed below you can find guides for interacting with the TRTL apps service.

HTTP endpoints

TRTL apps provides a set of RESTful API endpoints that your app can interact with. Most of these endpoints require authorization using a bearer token containing your app's secret key. You can find your app secret in the app console overview page.

You can find the full HTTP endpoints documentation here.

NPM package

If you are developing your app in a node.js environment, we highly recommend using the TRTL apps NPM package. The package is fully documented and provides strongly typed API function calls and many useful helper functions.

You can find the NPM package here, and the full API documentation here.
